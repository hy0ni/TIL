# 대소문자 변환하기

- 소문자를 대문자로 변환하기: toUpperCase()
- 대문자를 소문자로 변환하기: toLowerCase()

## toUpperCase()

주어진 문자열을 **대문자로 변환**해 반환함.

```javascript
const abc_uppercase = "abC".toUpperCase();
console.log(abc_uppercase); // ABC
```

## toLowerCase()

주어진 문자열을 **소문자로 변환**해 반환함.

```javascript
const abc_lowercase = "abC".toLowerCase();
console.log(abc_lowercase); // abc
```
