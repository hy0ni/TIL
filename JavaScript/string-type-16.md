# 문자열 숫자로 변환하기 (3가지 방법)

1. parseInt()
2. parseFloat()
3. +, \* 연산자

## parseInt() 함수 사용하여 문자열 숫자로 변환하기

```javascript
parseInt(string);
parseInt(string, radix);
```

`parseInt()` 함수는 문자열 인자를 파싱하여 특정 진수(수의 진법 체계에서 기준이 되는 값)의 정수를 리턴한다.  
string의 첫 글자를 정수로 변경할 수 없으면 NaN(Not a Number) 값을 리턴한다.

```javascript
parseInt("10"); // 10
parseInt("-10"); // -10
parseInt("10.9"); // 10
parseInt(10); // 10
parseInt("10n"); // 10
parseInt("10nnn13"); // 10
parseInt("    10"); // 10
parseInt("10      "); // 10
parseInt("k10"); // NaN
parseInt(""); // NaN
```

`parseInt("10")`  
문자열 "10"을 숫자로 변환하여 정수 10을 리턴한다.

`parseInt("-10")`  
문자열 "-10"을 숫자로 변환하여 정수 음수 -10을 리턴한다.

`parseInt("10.9")`  
문자열 타입의 실수값은 소수점을 제거한 후, 정수값만 리턴한다.

`parseInt(10)`  
파라미터로 문자열이 아닌 다른 타입의 값이 전달되면,
파라미터를 문자열로 변환하여 처리한다.

`parseInt("10n")`  
`parseInt("10nnn13")`  
문자열의 첫글자가 숫자이고, 그 이후에 숫자가 아닌 다른 문자열이 나올 경우
숫자가 아닌 문자 이후의 값은 무시하고,
그 이전의 숫자만 정수로 변환한다.

`parseInt(" 10")`  
문자열의 첫글자는 반드시 숫자여야 하지만, 처음에 오는 공백 문자는 허용된다.

`parseInt("10 ")`  
문자열의 첫글자가 숫자이면, 뒤에 오는 공백은 무시한다.

`parseInt("k10")`  
문자열의 첫글자가 숫자가 아니면, NaN(Not a Number)를 리턴한다.

`parseInt("")`  
문자열에 공백이 입력되면, 문자열의 첫 글자가 숫자가 아니므로, NaN(Not a Number)를 리턴한다.

## parseFloat() 함수 사용하여 문자열 숫자로 변환하기

```javascript
parseFloat(string);
```

parseFloat() 함수는 주어진 값을 필요한 경우 문자열로 변환한 후 부동소수점 실수로 파싱해 리턴한다.  
첫 글자가 숫자로 변환할 수 있는 값(숫자(0~9), 기호(+, -), 소수점(.), 지수)이 아닌 경우, NaN을 리턴한다.  
첫 글자 이후에 숫자로 변환할 수 없는 값이 오는 경우, 그 이후 값들은 무시한다.

```javascript
parseFloat("10"); // 10
parseFloat("-10"); // -10
parseFloat("10.9"); // 10.9
parseFloat(10.9); // 10.9
parseFloat("10.9abcd"); // 10.9
parseFloat("10.9abcd55"); // 10.9
parseFloat("abcd10.9"); // NaN
parseFloat("    10.9"); // 10.9
parseFloat("10.9   "); // 10.9
parseFloat(""); // NaN
```

`parseFloat("10")`  
입력받은 문자열에 소수점이하 자리가 없으므로, 그대로 정수 10을 리턴한다.

`parseFloat("-10")`  
음수 값도 변환할 수 있음.

`parseFloat("10.9")`  
문자열을 실수로 변환하여 실수를 리턴.

`parseFloat(10.9)`  
파라미터로 문자열이 전달되지 않고, 다른 타입의 값이 전달되면  
전달된 파라미터를 문자열로 변환한 후, 실수로 변환하여 그 값을 리턴한다.

`parseFloat("10.9abcd")`  
첫 글자 이후에 숫자로 변환할 수 없는 값이 오는 경우, 그 이후 값들은 무시한다.  
'abcd'는 숫자로 변환할 수 있는 값이 아니고, 첫 글자가 아니므로 무시된다.

`parseFloat("10.9abcd55")`  
첫 글자 이후에 숫자로 변환할 수 없는 값이 오는 경우, 그 이후 값들은 무시한다.  
10.9까지만 숫자로 변환할 수 있고, abcd는 숫자로 변환할 수 없음.  
55는 숫자로 변환할 수 있지만, abcd(숫자로 변환할 수 없는 값) 이후에 있으므로 무시된다.

`parseFloat("abcd10.9")`  
첫 글자가 숫자로 변환할 수 있는 값(숫자(0~9), 기호(+, -), 소수점(.), 지수)이 아닌 경우, NaN을 리턴한다.

`parseFloat("    10.9")`  
첫 글자가 공백인 경우, 공백은 무시함.

`parseFloat("10.9   ")`  
첫 글자 이후에 숫자로 변환할 수 없는 값이 오는 경우, 그 이후 값들은 무시한다.  
공백도 숫자로 변환할 수 없는 글자이므로, 무시됨.

`parseFloat("")`  
공백 문자만 입력된 경우, 변환할 수 있는 숫자가 없으므로 NaN을 리턴한다.

## +, \* 연산자 사용하여 문자열 숫자로 변환하기

### 단항 연산자 (+) 사용하기

문자열 앞에 단항 연산자 (+)를 붙여주면, 문자열을 숫자 타입으로 변환하여 리턴한다.  
만약, 문자열을 숫자로 변경할 수 없다면 NaN (Not a Number)를 리턴.

```javascript
console.log(+"10"); // 10
console.log(+"10.1"); // 10.1
console.log(+"-10"); // -10
console.log(+"abc"); // NaN
console.log(+true); // 1
console.log(+false); // 0
```

`+"10"`  
정수 10을 리턴.

`+"10.1"`  
실수 10.1을 리턴.

`+"-10"`  
음수도 처리 가능.

`+"abc"`  
숫자로 변경할 수 없는 문자열이기 때문에, NaN을 리턴.

`+true`  
`+false`  
boolean 값은 true인 경우 1, false인 경우 0을 리턴.

### 곱하기(\*) 연산자 사용하기

숫자로 변환할 문자열에 "곱하기 1" 을 해주면 문자열이 숫자로 변환된다.  
만약, 문자열을 숫자로 변경할 수 없다면 NaN (Not a Number)를 리턴한다.

```javascript
console.log("10" * 1); // 10
console.log("10.1" * 1); // 10.1
console.log("-10" * 1); // -10
console.log("abc" * 1); // NaN
```

`"10" * 1`  
정수 10을 리턴.

`"10.1" * 1`  
실수 10.1을 리턴.

`"-10" * 1`  
음수도 처리 가능.

`"abc" * 1`  
숫자로 변경할 수 없는 문자열이기 때문에, NaN을 리턴.
