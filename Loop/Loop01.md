자바스크립트에서 반복문은 특정 코드를 여러 번 실행할 때 사용됩니다. 가장 많이 사용되는 반복문으로는 for, while, do...while이 있습니다.

- 카운터(counter). 이는 어떤 값으로 초기화됩니다. 이것이 반복문의 시작점입니다.
- 조건(condition). 이는 반복문이 계속 실행될지, 혹은 멈출지를 결정하는 true/false 테스트입니다.
- 반복자(iterator). 이는 보통 조건이 더 이상 true가 아닐 때까지 각각의 연이은 반복에서 조금씩 카운터를 증가시킵니다. (조건이 더 이상 참이 아닌 지점, 따라서 반복은 종료됩니다)

## for 반복문

```javascript
// 문법(syntax)
for (초기화식; 종료 조건; 증감식) {
  // 실행할 코드
}
```

1. 괄호가 뒤따라오는 키워드 for.
2. 괄호 안에는, 세미콜론으로 구분되는, 세개의 항목이 있습니다:

- 초기화식(initializer): 보통 숫자로 설정된 변수인데, 반복문이 실행될 횟수(count)까지 증가됩니다. 이것은 또한 때때로 카운터 변수(counter variable)라고도 불립니다.
- 조건식(condition): 언제 반복문이 반복을 멈춰야만 하는지를 정의합니다. 일반적으로 비교 연산자(comparison operator)를 특징으로 하는 표현식(expression)인데, 이는 종료 조건이 충족되었는지를 확인하는 테스트입니다.
- 증감식(final-expression): 반복문이 전체 반복을 거쳤을 때마다 항상 평가(혹은 실행)됩니다. 보통 카운터 변수를 증가 (혹은 몇몇 경우에는 감소)시키기 위해, 조건이 더 이상 true가 아닌 지점에 가까워지게 하기 위해 제공됩니다.

1. 코드 블럭을 감싸는 중괄호: 반복문이 반복될 때마다 실행될 것입니다.

### 5번 반복하기

```javascript
for (let i = 0; i < 5; i++) {
  console.log(`반복 횟수: ${i}`);
  console.log(typeof i);
}
```

- let i = 0;: 초기화 부분, 반복 변수를 선언하고 초기값을 설정합니다.
- i < 5;: 조건식, 이 조건이 참일 때 반복이 실행됩니다.
- i++: 증감식, 반복 변수가 어떻게 변할지 지정합니다.

위 예시는 0부터 4까지 총 5번 반복됩니다.

위 코드에서 조건을 i <= 5가 아니라 i < 5로 작성했습니다.  
이유는 카운트가 1부터가 아닌 0부터 세기 때문입니다.

i < 5는 i가 0부터 시작하여 4까지의 값을 가지게 하여, 총 5회 반복합니다.  
i는 0, 1, 2, 3, 4가 됩니다. 이 경우, 5를 포함하지 않으므로 반복 횟수가 5회가 됩니다.

만약 조건을 i <= 5로 설정하면, i는 0부터 5까지의 값을 가지게 되어 총 6회 반복합니다.  
i는 0, 1, 2, 3, 4, 5가 되며, 5회 반복이 아니라 6회 반복됩니다.

## while 반복문

while 반복문은 조건이 참인 동안 계속해서 블록 내의 코드를 실행하는 반복문입니다  
반복 횟수가 정해져 있지 않거나, 조건에 따라 반복을 종료해야 할 때 사용합니다.

```javascript
// 문법(syntax)
while (종료 조건) {
  // 실행할 코드

  증감식
}
```

- 조건식 (Condition): 이 식이 true인 동안 반복문이 계속 실행됩니다. 조건식이 false가 되면 반복문이 종료됩니다.
- 반복 실행할 코드: 조건식이 true일 때 실행되는 코드 블록입니다.

```javascript
let i = 0;
while (i < 5) {
  console.log(`i의 값: ${i}`);
  i++;
}
/*
i의 값: 0
i의 값: 1
i의 값: 2
i의 값: 3
i의 값: 4
*/
```

i는 0부터 시작하여 5보다 작을 때까지 반복문이 실행됩니다.
각 반복에서 i의 값이 출력되고, i는 1씩 증가합니다.

## do...while

do...while 반복문은 while 반복문과 유사하지만, 조건식을 반복문 블록의 마지막에서 평가합니다.  
이로 인해 do...while 문은 반복문 블록이 최소한 한 번은 실행됩니다. 이 반복문은 주어진 조건이 true인 동안 반복됩니다.

```javascript
초기화식
do {
  // 실행할 코드

  증감식
} while (종료 조건)
```

- do 블록: 반복문이 시작되면 조건식이 평가되기 전에 우선 do블록 안의 코드가 실행됩니다.
- while 조건식: do 블록의 코드가 실행된 후, 조건식이 평가됩니다. true 
- 이 식이 true인 동안 do 블록의 코드가 반복 실행됩니다. 조건식이 false가 되면 반복문이 종료됩니다.


## break로 반복문 종료하기

만약 모든 반복이 완료되기 전에 반복문을 종료하고 싶다면, break문을 사용할 수 있습니다.
switch문에서 사용하는 break문과 동일하게,
break문은 조건이 충족되었을때 즉시 반복문을 종료하고 브라우저가 반복문 뒤에 있는 코드로 이동하게 합니다.

```javascript
for (let i = 0; i < 5; i++) {
  if (i === 2) {
    break; // i가 2일 때 반복문 종료
  }
  console.log(`반복 횟수: ${i}`);
}
/*
반복 횟수: 0
반복 횟수: 1
*/
```

위 코드에서 i가 2가 되면 break문이 실행되어 반복문이 종료됩니다.

## Continue로 반복 건너뛰기

continue 문은 반복문에서 특정 조건을 만족할 때 현재 반복의 나머지 부분을 건너뛰고 다음 반복으로 넘어갑니다.  
조건에 따라 특정 코드 블록의 실행을 건너뛰고 반복문을 계속하고자 할 때 유용합니다.

```javascript
for (let i = 0; i < 10; i++) {
  if (i % 2 === 0) {
    continue; // i가 짝수일 경우 현재 반복을 건너뛰고 다음 반복으로 넘어감
  }
  console.log(`홀수: ${i}`);
}
/*
홀수: 1
홀수: 3
홀수: 5
홀수: 7
홀수: 9
*/
```

위 코드는 i가 짝수일때 continue문이 실행되어 console.log를 건너뛰고 다음 반복으로 넘어갑니다.
